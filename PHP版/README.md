# PHP版: ブラウザ操作の ESC/POS 週次スケジュール印刷

PowerShell版 `m5-weekly-refill.ps1` を参考に、ブラウザ経由で操作するPHP版です。

## 主な機能
- 週テンプレート生成（週開始は月曜固定）
- 土日行の `!!...!!` マーカー（印字時は反転 ESC/POS）
- ESC/POS コマンド組み立て（初期化、日本語有効化、フォントB、行間、倍角、カット）
- TCPソケット送信（既定 `172.16.20.31:9100`）
- 週ごとのバックアップ保存/再利用（`m5-weekly-refill-YYYY-MM-DD.bak.txt`）
- 画面ログ表示（セッション内保持）

## 必要条件
- PHP 8.1+
- `mbstring` 拡張

## 起動方法（ローカル）
`PHP版` フォルダで以下を実行:

```bash
php -S 127.0.0.1:8080
```

ブラウザで以下を開く:

```text
http://127.0.0.1:8080/m5-weekly-refill.php
```

## 操作
- `開始日` を指定し、`<` / `>` / `週を反映` で週を切替
- 必要なら `プレビュー編集` をONにして編集
- `保存` で印刷前でも現在のプレビュー内容を週別バックアップとして保存
- `印刷` でプレビュー内容をそのまま送信
- `マーカー表示` OFF時は `!!...!!` を隠してプレビュー表示

## 補足
- 保存ファイルはスクリプトと同じフォルダに作成されます。
- `保存` と `印刷` のどちらでも、現在のプレビュー内容を週別バックアップとして保存します。
